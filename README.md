# Personal Task Website

This project is a lightweight personal to-do website built with plain HTML, CSS, and JavaScript.

## Features

- Add tasks with notes, due dates, and priority
- Create daily or weekly recurring routines with a custom reset time
- Drag tasks into a saved manual order
- Mark tasks complete
- Edit tasks and confirm before deleting them
- Keep recurring tasks in their own tab and completed one-off tasks in their own area
- Sort by manual order, priority, due date, or newest first
- Switch themes and appearance settings from the in-app settings panel
- Search tasks instantly
- Keep tasks saved in your browser with `localStorage`

## Run it

Open [index.html](./index.html) in your browser.

If you want a local server instead, from this folder you can run:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.
